# Titanic Data Analysis

This repository contains a complete **exploratory data analysis (EDA)** of the famous **Titanic dataset**, which includes passenger information such as age, gender, class, and survival status. The analysis aims to uncover patterns and insights about survival rates and factors that influenced them.

- Analysis of survival rates by **gender**, **passenger class**, and **age**.
- Visualizations including **bar charts**, **histograms** and **distribution plots**.
- Data cleaning and preprocessing steps to handle **missing values**.
- Insights on which features contributed most to passenger survival.

## Dataset
(https://www.kaggle.com/c/titanic/data)
